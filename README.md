# Emad Kamel Ayad - Modern Full-Stack Portfolio 🚀

A high-performance, visually stunning portfolio website built with the latest technologies. This project showcases a luxury aesthetic with glassmorphism, fluid animations, and a professional user experience.

## ✨ Features

- 🌓 **Dynamic Theming**: Advanced Light/Dark mode with seamless animated transitions and system preference detection.
- 📱 **Fully Responsive**: Optimized for all devices, from mobile to ultra-wide displays.
- 🎨 **Premium UI/UX**: Built with a luxury aesthetic using backdrop filters, oklch colors, and Framer Motion.
- 📧 **Resend Integration**: Secure, server-side contact form handling with professional email delivery.
- 🛍️ **Project Showcase**: A bold 2-column project grid with live demo and GitHub links.
- 📊 **Animated Stats**: Dynamic project and technology counters with custom emoji branding.

## 🛠️ Technology Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router & Turbopack)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Backend/API**: Next.js Route Handlers
- **Email Service**: [Resend](https://resend.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/) & [Lucide React](https://lucide.dev/)
- **State Management**: Zustand
- **Data Fetching**: TanStack Query (React Query)
- **Validation**: Zod & Typescript

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YoussefScript/portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file in the root directory and add your Resend API key:
   ```env
   RESEND_API_KEY=your_resend_api_key
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
